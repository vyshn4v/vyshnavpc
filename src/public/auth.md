# Auth.md

## Agent Registration
Agents can dynamically register for API access by using our OpenID Connect Discovery endpoints.
No manual human approval is required to access public portfolio data. To obtain client credentials, please use standard OAuth 2.0 Dynamic Client Registration.

This site does not currently require complex authentication for its public API operations.
To interact with the Contact API, you do not need an access token.
Agent registration is not required for public endpoints.
